# 🚀 E-commerce-LAWC

Este proyecto tiene como objetivo permitir a los usuarios **realizar compras de productos** mediante un sistema de carrito.  
Está desarrollado utilizando **JavaScript Vanilla**, siguiendo una arquitectura basada en **views**, **scripts** y **styles**.

## ✨ Características
- Agregar productos al carrito.
- Visualizar y gestionar el carrito de compras.
- Arquitectura modular para mejor mantenimiento.
